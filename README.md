# InVolt ⚡️

Aplicativo móvil nativo para el registro y gestión de lecturas de medidores eléctricos en zonas rurales. Proyecto MVP para **Infira**.

![InVolt Logo](assets/logo.png)

## Propósito
Digitalizar el proceso de toma de lectura de medidores en el centro poblado de Chetilla, Cajamarca. El sistema está diseñado para operar en condiciones de baja o nula conectividad (**Offline-First**).

## Características Principales
- **Registro de Lecturas**: Captura rápida de medidos.
- **Validación Fotográfica**: Captura de imagen del medidor como respaldo.
- **Geolocalización**: Registro automático de la ubicación de la lectura.
- **Sincronización Inteligente**: Sube los datos automáticamente cuando se detecta conexión.
- **Gestión de Clientes**: Base de datos local con información de usuarios y medidores.

## Estructura del Proyecto
- `mobile/`: Código fuente de la aplicación Flutter.
- `backend/`: Servicio de sincronización y gestión en Go.
- `assets/`: Recursos visuales y logos.
- `docs/`: Documentación técnica y especificaciones.
